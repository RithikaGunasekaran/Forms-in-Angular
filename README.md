## (Local Setup)

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Installation Steps

```bash
# 1. Install dependencies
yarn install
# OR
npm install

# 2. Start the development server
npm run start

# 3. Open your browser
The app will automatically open at http://localhost:3000
```
